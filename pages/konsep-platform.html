Aturan Reseller & Konsep Platform

1. Aturan Khusus Reseller Ebook Premium

User tidak otomatis boleh menjual ebook premium milik admin.

Hanya user dengan status:

RESELLER_AKTIF

Aturan Reseller:

  • Tidak boleh mengubah isi ebook
  • Tidak boleh mengubah harga tanpa izin admin
  • Tidak boleh mengklaim ebook sebagai milik pribadi
  • Tidak boleh membagikan file PDF mentah
  • Hanya boleh menjual melalui sistem platform
  • File ebook tetap disimpan di server admin

Jika melanggar aturan, status reseller akan dicabut tanpa pemberitahuan.

2. Struktur Role Sistem

Admin (Pemilik Platform)

  • Mengelola toko ebook premium
  • Mengelola semua user
  • Menentukan user sebagai reseller atau tidak
  • Mengelola semua transaksi
  • Mengatur harga dan komisi reseller

User (Pelanggan Sistem)

  • Memiliki toko sendiri
  • Boleh menjual produk digital milik sendiri
  • Tidak bisa mengelola toko admin
  • Tidak bisa menjual ebook premium admin tanpa izin

Reseller (User Khusus)

  • Semua hak user biasa
  • Boleh menjual ebook premium admin
  • Tidak memiliki akses ke file PDF asli
  • Hanya bisa menjual lewat sistem

3. Alur Sistem (Flow)

  1. User mendaftar akun
  2. User login ke dashboard
  3. User membuat toko digital
  4. User mengupload produk digital milik sendiri
  5. User mempromosikan toko mereka
  6. Pembeli melakukan checkout
  7. Sistem memverifikasi pembayaran
  8. Sistem memberikan akses download otomatis

Alur khusus reseller:

  1. Admin menyetujui user sebagai reseller
  2. Produk ebook premium muncul di toko reseller
  3. Reseller menjual produk sesuai aturan

4. Struktur Data (Logika Sistem)

Tabel Users

  • id
  • email
  • password
  • role (admin / user / reseller)
  • store_name

Tabel Products

  • id
  • user_id
  • title
  • price
  • file_path
  • is_premium_admin (true / false)

Tabel Orders

  • id
  • buyer_email
  • product_id
  • status (pending / paid)

Tabel Reseller

  • id
  • user_id
  • approved (true / false)

5. Tentang Platform

Platform ini menyediakan dua layanan utama:

  • Penjualan ebook premium milik admin
  • Sistem toko digital untuk user yang ingin menjual produk digital sendiri

Setiap user memiliki toko masing-masing yang bersifat mandiri dan dapat diakses secara publik. Produk yang dijual oleh user sepenuhnya merupakan tanggung jawab user tersebut.

Ebook premium milik admin tidak dapat dijual ulang oleh user kecuali telah disetujui sebagai reseller resmi.

Dengan sistem ini, toko admin dan toko user tidak bercampur dan berdiri secara mandiri.

6. Ringkasan

Admin menjual ebook premium dan menyediakan sistem toko digital. User hanya mengelola toko miliknya sendiri. Reseller adalah user khusus yang diizinkan menjual ebook premium admin sesuai aturan.
